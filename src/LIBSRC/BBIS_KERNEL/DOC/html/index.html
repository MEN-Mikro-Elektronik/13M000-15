BBIS kernel library for QNX Documentation

01.6

This is the documentation of the MEN BBIS kernel library for QNX. The BBIS kernel provides some basic services used by the MDIS kernel for opening and closing carrier board devices (like MEN A201, D203 etc.).

For QNX, the BBIS kernel library is compiled in two variants:

  • libmen_bbis_kernel.so: standard library, normally used
  • libmen_bbis_kernel_dbg.so: debug version
