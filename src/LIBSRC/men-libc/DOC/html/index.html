MEN C-library extensions for QNX6 Documentation

Release-1.6

This documentation describes the functions provided by the men-libc.a library. It containes supplementary routines developed by MEN.

The user application must be linked with libmen-libc.a, i.e. the Makefile or common.mk should contain the following lines:

  LIBS += men-libc
  EXTRA_INCVPATH += c:/menqnx/src/INCLUDE/COM c:/menqnx/src/INCLUDE/NATIVE
  EXTRA_LIBVPATH += c:/menqnx/$(OS)/$(CPUDIR)/usr/lib

Substitute c:/menqnx with the directory where the MEN BSP tree has been installed.

Additionally, your souce code should include the header file: 

  #include <MEN/men_libc.h>

System managment bus (SMB) functions

The SMB functions may be used to access devices on the SM bus. The BSP specific smb-server must be running in order to access the devices.

VMEbus functions

The VME functions can be used to access the (BSP specific) VME resource manager. It provides functions for:

  • VME Interrupt handling
  • Issue VME interrupts
  • Translate between VME and CPU addresses
